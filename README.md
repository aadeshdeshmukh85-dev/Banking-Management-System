
# 🏦 Banking Management System

## 📌 Project Overview

The **Banking Management System** is a full-stack web application designed to manage basic banking operations such as account creation, deposits, withdrawals, and debit card requests.

The system provides a simple and interactive user interface where users can perform banking transactions through a web browser. The backend is developed using **Spring Boot**, while the frontend is built using **HTML, CSS, and JavaScript**.

This project demonstrates the implementation of **REST APIs, database operations, and frontend-backend integration** in a banking application.

---

## 🎯 Objectives

* To develop a web-based banking management system
* To manage customer account details and transactions
* To perform basic banking operations such as deposit and withdrawal
* To demonstrate full-stack development using Spring Boot

---

### Frontend

* HTML

* MySQL

### Development Tools

* Eclipse IDE
* Web Browser (for localhost testing)

---

## ✨ Features

The system provides the following features:

* Create a new bank account
* Deposit money into an account
* Withdraw money from an account
* Apply for a debit card
* View all customer accounts
* Real-time account balance updates
* Simple and user-friendly interface
